# Getting started

# Prerequisites

  • Gitopen in new window
  • Node.jsopen in new window ^16
  • Yarnopen in new window >= 1.21.1
  • Supported OS: GNU/Linux, macOS and Windows

To install these prerequisites, you can follow the How To sectionopen in new window of the documentation.

# Install

# Clone the repository
$ git clone https://github.com/ovh/manager.git

# Go to the project root
$ cd manager

# If you are using nvm
$ nvm use

# Install
$ yarn install

# Allow your account to be used to develop on our open-source customer interfaces

In order for you to allow your account to connect to the applications we will describe in the Applicationsopen in new window, you will need to enable the "developer mode" on it.

To do so connect to the customer interface of the region in which your account has been created and go to the "Advanced settings" tab of your account configuration page.

  • EUopen in new window
  • CAopen in new window
  • USopen in new window

Once in there please check the box "I want to enable developer mode on this account"

Once clicked, the checkbox will look like this.

You can now follow the Applicationsopen in new window section of this documentation to try and boot your dev manager. You will be able to connect with your customer account to this dev version of the manager.
